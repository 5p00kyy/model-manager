CRITICAL BUGS FIXED
------------------
1. "Progress stuck at 'Preparing download...'"
   Root Cause: Widget updates from worker thread not triggering UI redraws
   Solution: Implemented Textual message passing system
   File: src/screens/download_screen.py

2. "Scaling changing sometimes"
   Root Cause: Rapid resize events causing flicker
   Solution: 200ms debounce on resize handler
   File: src/app.py

3. "Unable to see model name properly" / "Tables squish"
   Root Cause: Dynamic column widths causing truncation
   Solution: Explicit minimum widths for all columns
   Files: src/screens/main_screen.py, search_screen.py, detail_screen.py
